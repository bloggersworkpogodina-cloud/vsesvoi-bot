# Все свои CRM — V3.3.1 Referral Button

Обновление добавляет постоянную кнопку в нижнем меню бота:

- 🤝 Реферальная программа

Что работает:

- кнопка видна участнику в обычном меню Telegram;
- после регистрации бот дополнительно показывает постоянное меню с реферальной программой;
- по кнопке участник получает персональную ссылку;
- если участник уже зарегистрирован, ссылка строится на его последнее активное мероприятие;
- если регистраций нет и активное мероприятие одно — ссылка создается на него;
- если активных мероприятий несколько — бот предлагает выбрать мероприятие;
- показывается количество приглашенных по ссылке участника;
- админское меню тоже содержит кнопку 🤝 Реферальная программа.

Переменные Railway остаются прежними:

- BOT_TOKEN
- ADMIN_IDS
- DATABASE_URL
- GOOGLE_SHEET_ID
- GOOGLE_SERVICE_ACCOUNT_JSON
