# Vsesvoi Bot V2.1

Telegram-бот регистрации на мероприятия сообщества «Все свои».

## Что умеет

- `/admin` — админ-панель
- `/new` — создать мероприятие из Telegram
- `/events` — список мероприятий и ссылки
- `/link` — ссылка на последний ивент
- `/participants` — участники всех мероприятий
- `/participants EVENT_ID` — участники конкретного мероприятия
- `/summary` — ручная сводка по регистрациям

## Новое в V2.1

- Красивое уведомление админу после каждой регистрации.
- Счетчик всех регистраций на мероприятие.
- Счетчик регистраций за сегодня.
- Уведомление админу каждые 10 регистраций.
- Автоматическая сводка каждые 3 часа, если были новые регистрации.

## Railway Variables

Обязательные переменные:

- `BOT_TOKEN` — токен от BotFather
- `ADMIN_IDS` — Telegram ID админов через запятую

Опционально:

- `BOT_USERNAME` — username бота без @, например `vsesvoi_event_business_bot`

## Важно

Сейчас данные хранятся в JSON-файлах на Railway. Для долгосрочного использования лучше перейти на PostgreSQL, чтобы данные не терялись при пересборке сервиса.
