# Vsesvoi Bot V2.3 CRM Menu

Добавлено:
- постоянное нижнее меню администратора;
- новая CRM-главная `/admin`;
- разделы: Главная, Мероприятия, Участники, Аналитика, Рассылки, Настройки;
- карточка мероприятия с кнопками: ссылка, участники, статистика;
- просмотр участников конкретного мероприятия;
- базовая аналитика по всем мероприятиям;
- сохранены юридический модуль, регистрация, `/delete_me`, сводки и создание мероприятий.

## Переменные Railway
- BOT_TOKEN
- ADMIN_IDS
- BOT_USERNAME (опционально)

## Запуск
Railway запускает через Procfile:
`worker: python bot.py`
