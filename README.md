# Все свои CRM Bot — V2.2 Legal

Добавлено:
- юридический экран перед регистрацией;
- 4 документа внутри Telegram;
- кнопка «Ознакомился и принимаю условия»;
- фиксация версии документов и времени принятия;
- новые поля регистрации: имя, фамилия, телефон, сфера деятельности, город;
- команда `/delete_me` с подтверждением удаления данных;
- уведомление администратору при удалении данных.

Переменные Railway:
- `BOT_TOKEN`
- `ADMIN_IDS`
- `BOT_USERNAME` опционально
