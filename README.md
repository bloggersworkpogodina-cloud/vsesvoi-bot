# Хранитель чудес

Telegram-бот для чата «Личный отдел чудес».

## Что делает

- удаляет системное уведомление о вступлении в чат;
- приветствует нового участника и отмечает его кликабельным именем;
- просит рассказать о себе;
- даёт кнопку «📜 Правила чата»;
- правила открываются в личке с ботом;
- приветствие автоматически удаляется через 10 минут.

## Railway Variables

Обязательные переменные:

```env
BOT_TOKEN=токен_бота
BOT_USERNAME=otdel_chudes_bot
```

Необязательная переменная:

```env
DELETE_WELCOME_AFTER_SECONDS=600
```

## Важно

Бот должен быть администратором чата с правом удалять сообщения.
