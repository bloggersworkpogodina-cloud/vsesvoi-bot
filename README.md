# ВСЕСВОИ Telegram Bot — V2

Функции:

- регистрация по ссылке конкретного мероприятия;
- создание мероприятий через Telegram командой `/new`;
- `/admin` — меню администратора;
- `/events` — список мероприятий и ссылки;
- `/link` — ссылка на последнее мероприятие;
- `/participants` — все участники;
- `/participants event_id` — участники конкретного мероприятия.

## Railway Variables

Обязательные переменные:

- `BOT_TOKEN` — токен из BotFather
- `ADMIN_IDS` — Telegram ID админа, например `372122727`

Опционально:

- `BOT_USERNAME` — username бота без @, например `vsesvoi_event_business_bot`

## Как создать мероприятие

В Telegram напишите боту:

```text
/new
```

Дальше бот по шагам спросит название, дату, время, описание, ссылку на чат и канал.

Если хотите использовать стандартный чат/канал, на вопрос про ссылку отправьте:

```text
-
```

После подтверждения бот даст ссылку регистрации.
