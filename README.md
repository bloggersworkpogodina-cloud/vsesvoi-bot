# Все свои CRM — V3.3 Referral Program

Стабильная версия Telegram CRM для мероприятий.

## Что есть

- Создание мероприятий
- Уникальные ссылки регистрации
- Юридический блок и согласия
- Регистрация участников
- PostgreSQL как основное хранилище
- Google Sheets как автоматическая таблица регистраций
- CRM-меню администратора
- Карточки мероприятий
- Карточки участников
- Рассылки по базе и по мероприятию
- Реферальная программа V1

## Новое в V3.3

- После регистрации участник получает персональную реферальную ссылку
- Если новый участник приходит по реферальной ссылке, CRM сохраняет, кто его пригласил
- В карточке мероприятия добавлена кнопка «🤝 Рефералы»
- В CRM показывается рейтинг участников по приглашениям
- В Google Sheets добавлены колонки:
  - Пригласил Telegram ID
  - Пригласил username
  - Реферальная ссылка

## Railway variables

Обязательные:

- BOT_TOKEN
- ADMIN_IDS
- DATABASE_URL

Для Google Sheets:

- GOOGLE_SHEET_ID
- GOOGLE_SERVICE_ACCOUNT_JSON

Дополнительно:

- BOT_USERNAME

## Проверка после деплоя

1. Открыть `/admin`
2. Открыть карточку мероприятия
3. Проверить кнопку «🤝 Рефералы»
4. Пройти регистрацию
5. Убедиться, что после регистрации бот выдал персональную ссылку
6. Зарегистрировать второго человека по этой ссылке
7. Проверить рейтинг рефералов
